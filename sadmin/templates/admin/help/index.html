使用说明

一、引言

本使用说明文档介绍了驾校管理系统的开始模块、教练模块、车辆模块、学员模块、驾校模块的功能及操作使用方法。
本使用说明文档是为指导驾校管理人员使用“嘻哈学车驾校管理系统”而编写，希望该文档使他们在使用系统过程中能起到无师自通的作用。
该系统由安徽嘻哈网络技术有限公司开发。使用者是驾校管理人员、驾校部门领导及有关人员，使用时各驾校必须使用相应的登录账号和登录密码才可登录各驾校的驾校管理系统进行管理操作。

二、运行环境

应用系统

操作系统：PC：windows XP，win7，win8

网络要求：必须链接网络

硬件要求

台式电脑或者笔记本一台（需要连接网络），建议驾校使用台式电脑，并且设置管理员账号和密码。防止无关人员修改或者查看驾校管理系统信息。

三、系统用途

    安全保密

  1. 系统具有较好的安全保密机制，各驾校在使用系统前必须先登录，依自己的登录账号和登录密码使用系统。

    2. 模块功能

  2. 教练模块
    直接管理教练，实时查看教练接单信息及学员对教练的评价。

  3. 车辆模块
    实时查看和更新车辆信息，车辆运行轨迹监控回放，设备状态信息监控。

  4. 学员模块
    实时查看学员订单时间，价格，数量，订单状态实时掌控。

  5. 驾校模块
    实时更新驾校信息，更改添加班制信息，更新报名点信息。

四、使用方法

    驾校登录

  1. 在浏览器中输入http://www.xihaxueche.com进入嘻哈学车官网，然后即可直接进入登录界面

    2. 操作方法

  2. 教练模块操作方法简述
    可在教练模块中手动添加驾校教练，删除教练。点击修改可以实现修改教练基本信息，教练是否在线，教练教学时间配置等操作。


  3. 车辆模块操作方法简述
    点击添加车辆，可加入新的车辆信息。点击修改，可更改已加入的车辆信息，点击删除可删除已加入的车辆信息。

  4. 学员模块
    点击添加订单可手动添加订单，点击学员档案，可查看在本校学车的学员详细信息。

  5. 驾校模块操作方法简述
    点击编辑驾校可修改驾校的基本信息，点击轮播图管理可随时添加更改轮播图，
    点击班制管理可修改和添加班制信息，点击报名点管理可修改和添加驾校报名点的信息。
