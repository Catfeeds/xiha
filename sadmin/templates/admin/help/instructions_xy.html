学员管理


  1. 如上图所示，点击进入订单管理界面，首先在左侧侧边栏选择所需查看的订单的班制类型。

  2. 点击左侧绿色添加订单即可手动添加新订单。

  3. 搜索选择框中选择搜索条件：学员姓名/号码,订单号,及支付方式。填写搜索内容即可搜索相应订单

  4. 将鼠标放在订单号上或者点击订单号可查看订单详情。

  5. 右边设置按钮可设置订单状态：已支付，未支付，取消报名。

  6. 点击编辑按钮可手动编辑报名驾校订单学员的信息。



如上图所示，点击进入编辑报名订单界面，系统默认原订单信息，管理员可根据实际情况对报名订单进行相应的修改。



  1. 如上图所示，点击左侧学车订单管理，左侧侧边栏选择所需查看的学车订单的订单状态。

  2. 点击左侧绿色添加订单即可手动添加新订单。

  3. 搜索选择框中选择搜索条件：学员姓名/号码,教练号码，订单号,及支付方式。填写搜索内容即可搜索相应订单

  4. 将鼠标放在订单号上或者点击订单号可查看订单详情。

  5. 右边设置按钮可设置订单状态：已完成，未完成，已取消。

  6. 点击编辑按钮可手动编辑预约学车订单信息。



  1. 进入学车订单编辑界面，首先点击自动生成订单号，录入学员手机号，点击学员姓名录入框即可自动获取学员姓名。

  2. 选择预约学车教练的手机号，也可点击查看该教练的时间配置。

  3. 选择预约学车时间，点击加号按钮可添加预约学车时间，最多添加2次。

  4. 支付方式可选择线上或者线下支付，填完信息后提交即可完成对预约学车订单的编辑。




  1. 学员档案管理和档案回收站为报名驾校订单和预约学车订单学员的档案管理模块。

  2. 如图中所示，驾校管理员可手动添加新的学员档案，也可编辑修改老学员的档案信息。

  3. 档案回收，档案还原，档案删除。
    驾校可将无用的学员档案放置档案回收站，但学员档案信息还可以在档案回收站中保留，在档案回收站中还可以还原该档案信息；但是，如果管理员在档案回收站站中删除档案信息，那么该学员的档案信息将会彻底被删除了。
